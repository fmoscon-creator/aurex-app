Onboarding

Cómo funciona AUREX.

Tocá cada solapa para ver qué podés hacer adentro.

💼Portfolio▾
  • ➕Agregá activos: botón + arriba a la derecha del header.
  • 💰Valor total y moneda: tocá el badge $ para ciclar entre USD / USDT / BTC. PnL del período al lado.
  • ⏱️Períodos: 24h / 7d / 1m / 3m / 1y debajo del valor total.
  • 🌡️Termómetro de riesgo: ALCISTA / BAJISTA / ALTA-CONV-IA / SIN SEÑAL.
📊Mercados▾
  • 🗂️Categorías: Cripto / Acciones / Stable / Futuros / Commodities / Metales / ETFs / Bonos / Forex.
  • 🌍Países (en Acciones): USA / Argentina / Brasil / Europa / España / Japón / China.
  • ⏱️Temporalidades por activo: 24h / 7d / 1m / 3m / 1a.
  • 🔍Buscar: lupa arriba a la derecha.
👀Watchlist▾
  • ➕Agregar activos: tocá el + arriba a la derecha del header.
  • ⭐Crear varias listas: tocá el ícono Watchlist arriba a la derecha. Renombrar, color y agregar nuevas.
  • ➕Agregar lista nueva: botón "+ Nueva lista".
  • Comparador: seleccioná 2 o 3 activos y tocá "Comparar".
🤖Señales IA▾
  • 📋Lista de señales activas: TODAS las del motor IA al toque.
  • 🔀Filtros: por tipo, por convicción IA (%), por categoría de activo.
  • 💼Vista Portfolio: solo las señales sobre activos que ya tenés.
  • 👆Tocá cualquier señal: explicación detallada con las 24 variables que la generaron.
🔔Alertas▾
  • 📡15 tipos en total: IA / Pulse / Precio / Eventos macro y geopolíticos.
  • 🗂️Activar/desactivar por tipo: switches arriba.
  • 🤖Alertas IA: cambio de señal, alta convicción 85-88%, umbral propio, señal en Portfolio.
  • 💓Alertas Pulse: cambio de zona de mercado, termómetro global, categoría.
👤Perfil▾
  • 🌙Modo claro/oscuro: cambia el tema visual de toda la app.
  • 🌐Idioma: 8 disponibles (EN/ES/PT/ZH/FR/IT/HI/AR).
  • 🔒Cambiar contraseña.
  • 📩Contacto: enviá un mensaje al equipo AUREX.
