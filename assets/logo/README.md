# AUREX — Logo Oficial

## Logo aprobado
Fecha de aprobación: 27/03/2026

El logo oficial de AUREX consiste en:
- **3 círculos concéntricos dorados**: exterior grueso (3.5px), medio fino (1.5px), interior suave (1px opacity 0.8)
- **Letra A dorada** con barra horizontal, trazos redondeados
- **Chevron dorado** (flecha hacia abajo) debajo de la A, con punto central
- **Gradiente dorado**: de #F7D060 (claro) a #D4A017 (medio) a #8B6208 (oscuro)

## Archivos en esta carpeta

| Archivo | Uso |
|---|---|
| `aurex_logo_transparent.svg` | Logo oficial — fondo transparente. Usar sobre fondos oscuros |
| `aurex_logo_dark.svg` | Logo sobre fondo oscuro (rect con relleno radial marrón) |

## Uso en el código

El logo se usa **inline como SVG** en index.html con IDs únicos por instancia:
- `gAo1 / gCo1` — instancia OBS1 (120x120px)
- `gAo2 / gCo2` — instancia OBS2 (70x70px)  
- `gAhd / gChd` — instancia Header (32x32px)
