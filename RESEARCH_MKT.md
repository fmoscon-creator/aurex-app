## Plantilla por competidor

Para cada uno, en este orden estricto:

1. **Posicionamiento** — frase de cómo se vende ("X para Y que necesitan Z")
2. **Audiencia objetivo declarada** — principiantes / experimentados / ambos
3. **Modelo de monetización** — FREE+PRO / comisiones / suscripción / pago único
4. **Canales de marketing activos** — qué redes/plataformas usan
5. **Tipo de contenido por canal** — con 2-3 ejemplos concretos por red
6. **Tono y voz** — formal / cercano / técnico / educativo / irónico (con cita)
7. **Cómo se lanzaron** — timing, primeros pasos, pre-lanzamiento
8. **Qué les funcionó para crecer** — referrals / virality / content / ads / PR
9. **Reseñas App Store y Google Play** — qué dicen los usuarios (pros y contras)
10. **Fuentes consultadas** — URLs con fecha de consulta
